  ## 프로젝트 소개
  ‘COEARTH’는 위치기반 제로웨이스트 샵 검색 어플이다. 공유, 공동의 의미인 ‘CO-와 지구의 ’EARTH’의 합성어로, 우리 함께, 지구와 함께 착한소비를 실천하자는 의미로 어플의 이름을 정했다. 사용자의 주변 뿐만아니라 지역별 제로웨이스트샵도 검색이 가능하며, 제로웨이스트 샵 정보, 별점, 리뷰 등 정보를 자세히 볼 수 있다. 리뷰 작성을 통한 사용자 간 정보 공유와 실제 방문한 사용자의 후기를 통해 방문하기 전, 정보를 얻을 수 있다. ‘네이버지도 API’와 ‘Firebase’를 활용하여 지도와 DB, 사용자 정보 DB를 구현하였다. 이 앱의 목표는 자신과 가까운 곳에서 쉽고 간편하게 지구를 위한 착한 소비를 유도한다.
